円柱の回転運動

説明

円柱の自由回転運動のシミュレーターです。運動方程式の厳密解を使って計算しています。

  • シミュレーション画面上でマウスを左ボタン、中ボタン、右ボタンでドラッグすることにより視点を色々と動かすことが出来ます。
  • 細長い円柱にして、傾きを90度にした時が、体操の捻りなしの宙返りです(地面が縦向きになっていると考える)。90度より少し傾きを小さくした時が、捻りを伴った宙返りに相当します。捻りシミュレーターは、これよりもっと体操寄りにしたシミュレーターです。
  • 「時の流れ」で運動をスローで再生するか早送りで再生するかを変えられます。これは、物理的なパラメーターではありません。
  • 「角速度と角運動量」を押すと、角速度が黄色で、角運動量が緑色で表示されます。自由回転なので、角運動量は保存されているため動きません。
  • 角速度や各種楕円体などが円柱の中に見辛いとき、「原点をずらす」を押すと、これらが円柱に重ならないようにして表示されるようになります。
  • 角運動量保存則と回転のエネルギーから始まる一連の記事に、色々と詳しく書いています。
  • 「節線」は、オイラー角の文脈で使われる言葉です。もし、円柱が"嘘の回転"、つまり、対称軸を軸とする回転が無視され、対称軸の回転(歳差運動)しかしない場合、円柱の赤線がどこに来るか、を表しています。Bullet physicsや、Blender game engineなどの物理エンジンでは(私が調べた時点では)、剛体の回転を、この嘘の回転でごまかしています。これでは、体操の捻りを伴った宙返りはシミュレート出来ません。
  • 半径や高さを変えて、細長い円柱にしたり、平たい円柱にしたりして、回転運動がどうなるか見て下さい。節線を表示して、節線と赤線どちらがどちらを追い越すかなどが見所です。高さが半径の√3倍になるとき(対称軸回りの慣性モーメントと、対称軸に垂直な軸の慣性モーメントが等しくなる)が運動の変わり目で、この時、各種の楕円体が球になります。
  • 角速度ベクトルの指す延長上に慣性楕円体と不変平面の接点があること、角運動量ベクトルの先端がBinetの楕円体と角運動量一定の球面の交わる線上に乗っていることなどを見て下さい。
